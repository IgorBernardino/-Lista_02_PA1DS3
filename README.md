Estruturas de Repetição · Vetores · Matrizes em **C++**.
Dupla: **Júlia** e **Igor de Santana Bernardino**.

Mesmo visual da Lista 01 (`lista01.css`), com **enunciado · código C++ · print do VS Code**
para cada um dos 18 exercícios — sem pseudocódigo e sem fluxograma.

## Estrutura
```
.
├── index.html      ← a página (abra para ver)
├── lista01.css     ← o estilo (precisa estar na mesma pasta do index.html)
├── image/          ← prints do VS Code (exe01.png ... exe18.png)
└── codigos/        ← os 18 fontes .cpp (JuliaIgor001.cpp ...)
```

## Publicar no GitHub Pages
1. Suba estes arquivos para a raiz de um repositório no GitHub
   (mantenha `index.html` e `lista01.css` juntos na raiz).
2. **Settings → Pages → Branch: `main` / `(root)` → Save**.
3. O link sai como `https://SEU-USUARIO.github.io/NOME-DO-REPO/`.

## Prints
Tire o print de cada código rodando no VS Code e salve em `image/` como
`exe01.png`, `exe02.png`, ... `exe18.png`. A página exibe sozinha; enquanto
não houver imagem, aparece o espaço reservado.
